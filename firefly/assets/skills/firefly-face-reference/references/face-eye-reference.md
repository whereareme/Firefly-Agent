# Firefly Face And Eye Reference

Use this reference sheet with `../assets/firefly-face-reference.png`.

## Face

- Soft oval youthful-adult anime face.
- Small, clean nose.
- Tiny gentle closed-mouth smile.
- Calm, warm, clear expression.
- Fine eyebrows with a soft arc.
- No heavy makeup, sharp mature glamour, exaggerated lips, or childlike round face.

## Eyes

- Large but not childlike round eyes.
- Upper eyelid line is slightly dark blue-violet and softly curved.
- Iris outer ring: blue-violet to cyan / teal-blue.
- Iris interior: clear glassy cyan-blue highlights.
- Center pupil: vivid rose-pink / magenta vertical oval.
- Lower iris: pink-lavender to soft pink-purple gradient.
- Highlights: glossy white and cyan glass reflections.
- Avoid plain blue eyes, green eyes, black pupils, brown eyes, or purple-only eyes.

Compact prompt phrase:

```text
blue-violet/cyan outer iris ring + vivid rose-pink/magenta oval center pupil + pink-lavender lower iris + glossy white/cyan glass highlights
```

## Hair And Head Details

- Silver-white long hair with soft gray shading.
- Thick center bangs fall between the eyes without fully covering them.
- Smooth side locks frame the cheeks.
- Black or dark gray headband across the top of the head.
- Teal triangular accent on the headband/front hair area.
- Pale green translucent leaf-like side ornament, often with dark ribbon detail.

## Role Assignment For Multiple References

When another image is supplied:

- Use `assets/firefly-face-reference.png` for face, eyes, hair, and identity.
- Use the new user image only for pose, clothing, room, camera, color mood, or composition.
- State this role assignment explicitly in the prompt.

Example:

```text
Input image 1 controls Firefly's face and eyes. Input image 2 controls only the outfit, pose, and composition.
```
