# 初识篇完整资源 v2

这套资源是重新补全后的版本：场景先生成完整画幅，表情则分别生成为独立的完整立绘，不再把拼板条带直接当成成品。

- `characters/expressions/`：五张独立的完整腰部以上表情立绘。
- `scenes/`：四张完整的日常背景画幅。
- `cg/chapter-climax-sunset.png`：独立完整的章节收束 CG。
- `source/`：对应的完整生成母版，方便后续重新导出。
- `manifest.json`：剧情引擎使用的稳定资源 ID 和路径。

表情立绘目前带统一浅色背景，适合第一版剧情视图的矩形角色区域。需要叠加到任意场景时，再做专门的透明抠图版本；不要重新裁切这些完整文件。
